# Unnantended Test

API to provide the functionalities to create a playing card games with these specifics endpoints

- `/create-deck`
- `/open-deck`
- `/draw-card`

## Dependencies

* Go 1.16.3

## Build project
```console
$ unzip <project_folder>
$ cd <project_folder>
$ sh build.sh
```

## Run tests
```console
$ cd <project_folder>
$ sh test.sh
```

## Start the server
```console
$ cd <project_folder>
$ sh start.sh
```
* Open the browser on `localhost:9091`
